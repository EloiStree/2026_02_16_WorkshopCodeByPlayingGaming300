
Salut a toi ;)

J ai creer un atelier pour Unity 3D qui s appelle de Zero a NES.

L idee est d apprendre la programmation en creant une manette NES avec les UI du game engine et les touches de claviers / manettes.
Une fois la manette NES creer, on essai de jouer as des jeux  et on essayer de faire un manette Xbox pour repratiquer le exercie.

Il faut re-pratiquer pour que notre cerveaux assimile bien la matiere.

La video est ici pour la manette NES sous Unity:
https://youtu.be/dLRaB3z-4DQ

Le but de l exercice ici est de faire pareille mais avec Godot.

Comme vous ne savez pas encore ce qu est de la programmation.
